# Full Level Pass
## Problem
The draft keeps bumping into the same issue because it has four related levels in play:

1. **Semantic/conceptual rules:** application conditions, correctness conditions, modal rules.

2. **Discourse roles:** what expressions and claims do in use, such as mark commitments, license contrasts, organize inquiry, or repair disagreement.

3. **Field/inquiry roles:** what concepts and categories do inside ordinary practice, science, metaphysics, and other stabilized fields.

4. **Projectibility tests:** whether a category supports projections, failure conditions, support grades, and demotion rules.


Some sentences name only the first level (`semantic`) or only the third (`inquiry`), even when the local argument needs the broader sequence. The pass should make the sequence explicit and stable.
## Controlling Formula
Use this as the paper's recurring map:

> Rules of use govern correct application in discourse; discourse roles explain what expressions and claims do in a practice; field-inferential roles explain what categories license within an inquiry; projectibility tests whether those licensed inferences are preserved or defeated by the world.
## Edits To Make
1. **Abstract:** Split or tighten the abstract so it stops doing all four levels in one overlong paragraph. Make the opening sentence name rules of use, discourse roles, and field-inquiry roles without sounding like `semantic` has been dropped.

2. **Introduction:** Revise the first paragraph so `semantic and conceptual rules` isn't isolated from discourse. Better: modal claims express rules of use in the object language; those rules are semantic/conceptual, but their work is discourse-practical.

3. **Function paragraph:** Keep the three-level definition, but make it less ad hoc. The order should be: semantic/conceptual, discourse, field-inferential/projectibility.

4. **OME paragraph:** Keep the current correction: application conditions are rules for competent use in discourse, not stated definitions or mere evidence conditions.

5. **NN paragraph:** Change `At the semantic level` if it makes modal discourse sound merely semantic. Prefer: `At the level of modal discourse` or `For metaphysical modal discourse`.

6. **Convergence paragraph:** Include `use, discourse, inquiry, and revision` consistently.

7. **SFL section:** Make clear that SFL is dispensable because the needed claim is functional pluralism about discourse, not because discourse drops out.

8. **Residual disagreement:** Make ordinary linguistic practice consistently a field of discourse-practice, not simply `ordinary use` in a way that hides discourse.

9. **Conclusion:** Mirror the introduction's four-level map: rules of use, discourse roles, field-relative category choice, and projectibility discipline.

## Guardrails
- Don't reintroduce `opponent`, `unmet`, `target`, `bridge`, or `displaced`.

- Don't use `class`, `classify`, or `classification` technically.

- Keep `defeat/defeater` only for projection failure or ordinary defeat conditions, not dialectical victory.

- Preserve the field/jurisdiction/testing metaphor.

- Don't add new citations unless a source-grounded claim genuinely requires one.
