# House Style Snapshot

This directory vendors the central Brett Reynolds house style files needed to build the public repository as a standalone clone.

Included files:

- `preamble.tex`: LaTeX preamble and semantic macros.
- `style-rules.yaml`: machine-readable central house style rules.
- `VERSION`: central house style version at the time of vendoring.

The source of truth for active writing remains the central house style in the parent `LLM-CLI-projects` workspace.
